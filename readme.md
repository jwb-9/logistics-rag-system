
## 功能特性

- **RAG智能问答**：基于检索增强生成技术，提供精准的物流知识问答
- **工具调用**：支持计算器、网页访问等工具扩展能力
- **对话管理**：支持多轮对话，保持上下文记忆
- **Web界面**：提供友好的Web聊天界面
- **WebSocket支持**：支持实时消息推送
- **容器化部署**：支持Docker一键部署

## 技术栈

- **框架**: FastAPI
- **语言模型**: 支持多种LLM（需配置API密钥）
- **向量数据库**: Chroma
- **前端**: HTML + CSS + JavaScript
- **部署**: Docker / Docker Compose
